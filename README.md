Кредитный калькулятор для расчета графика ежемесячных платежей по кредиту. 
Программа принимает на вход сумму кредита, процентную ставку, тип кредита (аннуитетный или дифференцированный) и срок в годах, рассчитывает и отображает график платежей.

В программе были добавлены ограничения ввода данных пользователем, для исключения некорректных значений. 
Реализован расчет нового финансового показателя – суммы переплат.
Добавлен новый функционал – расчет кредитного графика при наличии первоначального взноса при аннуитетном графике выплат.
